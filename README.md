# Welcome Dammusi

Guide de bienvenue pour les invités des Dammusi Kuddia Attalora à Pantelleria.

## Deux pages

- 🏠 **i Dammusi** : informations pratiques (arrivée, eau, cuisine, wifi...)
- 🏝️ **L'Isola** : recommandations sur l'île (restaurants, plages, activités...)

Contenu bilingue FR / EN.

## URLs

- **Publique** (invités) : `/index.html`
- **Éditeur** (admin uniquement) : `/editor.html`

## Fonctionnement

Le contenu par défaut est embarqué dans `index.html` (toujours visible même sans connexion à jsonbin).
L'éditeur permet de modifier le contenu via jsonbin ; les modifications prennent le pas sur le contenu par défaut.

## Sécurité

- Viewer public : clé jsonbin en lecture seule
- Éditeur : clé maître (URL non partagée)
